# Tricolor Paulista 🏆🔥

Bem-vindo ao **Tricolor Paulista**, um site dinâmico e automatizado para acompanhar as últimas notícias, estatísticas e informações sobre o São Paulo FC.

## 📌 Funcionalidades  
✔️ Notícias atualizadas automaticamente  
✔️ Layout responsivo e otimizado  
✔️ Integração com IA para análises esportivas  
✔️ Alto desempenho com carregamento rápido  

## 🚀 Tecnologias utilizadas  
- **HTML5** → Estrutura da página  
- **CSS3** → Estilos visuais e responsividade  
- **JavaScript** → Interatividade e automação de notícias  
- **Cohere AI** → Geração de análises automatizadas  
- **GitHub + Vercel** → Hospedagem do projeto
